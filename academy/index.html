WordPress Website Maintenance

Training Course

Introduction

WordPress is a free, open source publishing software that can be installed locally on a web server and viewed on a proprietary web site or hosted in the cloud and viewed on the WordPress web site.

This course aims to teach you the basics of WordPress and to get you started in building your personal or business website.

Course Content

Introducing WordPress

You will discover the basic concepts about publishing a website with WordPress. The types of WordPress websites are also introduced.

Setting Up WordPress

This section explains how to setup a WordPress website. This process includes choosing a web hosting provider to "serve up" your website.

Extending WordPress

In this section you will learn how to:

  1. Use the WordPress Media Library to manage images and media.
  2. Embed videos in your content to provide media for your readers to interact with.
  3. Discover, download, and install plugins for WordPress to extend the features available on your website.
  4. Find and install WordPress themes that give your website a great visual look.
Customizing WordPress

In this section you will learn:

  1. All about the basic templates used to make up a WordPress theme, including basic template tags.
  2. How to utilize basic HTML markup and CSS to tweak the look and feel of your existing theme (optional).
  3. The basic concept behind Parent/Child themes in WordPress (optional).
