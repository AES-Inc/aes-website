WordPress 101 - The Complete Guide

Learn how to build your WordPress website from an experienced WordPress developer

Do you want to learn how to build your own website even if you have no prior experience or programming and coding knowledge?

This course will help you get started, whether you're looking to create a WordPress website for personal use or for your business. You'll start with the installation of WordPress. Progressing through the sections, you'll explore how to create a custom webmail account. To build on your skills, you'll even get to grips with configuring various WordPress plugins. In addition to this, the course covers the latest features in the WordPress 5.5 version update that will help you to further enhance your web page.

By the end of this course, you'll have gained a comprehensive understanding of WordPress and have learned how to confidently build a WordPress website through expert guidance.

All resources and supporting files for this course are available on GitHub at https://github.com/PacktPublishing/WordPress-for-Beginners-The-Complete-2020-WordPress-Guide

Who this course is for

This course is for anyone without prior programming experience who is looking to build a website for personal use or for business purposes.

What you will learn from this course

  • Install WordPress either locally on your computer or live on the internet
  • Install SSL encryption on your website
  • Create a professional webmail account
  • Work with themes and understand how to choose the right one
  • Create and edit image galleries
  • Publish content with pages and posts
  • Understand how to create user accounts
  • Create privacy and policy pages
  • Build your mailing list using Mailchimp
  • Discover how to back up your WordPress website in case of an emergency
  • Set up comments and contact pages
  • Prevent spam comments
  • Speed up your website with caching plugins
  • Export and import content effectively
